# Video to Product

## Vision
Персональный инструмент для превращения видео (YouTube и загруженных файлов) в структурированные продукты. Пайплайн: видео -> транскрипт -> саммари -> продукт (системный промпт для агента, чек-лист, курс). Для личного использования.

## Problem
Нет системного процесса извлечения знаний из видео и упаковки их в готовые продукты. Сейчас это делается вручную или не делается вообще.

## Target user
Татьяна — предприниматель, которая смотрит экспертные видео и хочет превращать их в рабочие инструменты (агенты, чек-листы). 2-3 видео в неделю.

## Core features (MVP)
1. Ввод YouTube URL или загрузка видео/аудиофайла -> извлечение аудио -> транскрипция через OpenAI Whisper API
2. Из транскрипта -> структурированное саммари через Claude API
3. Из саммари -> генерация продукта по шаблону "системный промпт для агента" с редактором в интерфейсе

## User journey
Открывает веб-интерфейс -> вставляет YouTube URL или загружает файл -> ждет транскрипцию -> видит саммари -> нажимает "создать агента" -> редактирует результат в textarea с превью -> копирует/сохраняет

## Business model
Персональный инструмент, не монетизируется напрямую. Создает ценность через продукты (агенты, материалы для клиентов).

## Tech stack
- Python 3.12, uv
- FastAPI + Jinja2 (веб-интерфейс)
- yt-dlp (скачивание аудио с YouTube)
- OpenAI Whisper API (транскрипция)
- Anthropic SDK (саммари и генерация продуктов)
- Без базы данных (одноразовый конвейер, без хранения истории)
- VPS (155.212.226.75), отдельный порт, nginx

## Data model (sketch)
Без персистентного хранения. Все данные живут в рамках сессии:
- Транскрипт (текст)
- Саммари (текст, markdown)
- Продукт (текст, markdown)

Временные файлы (скачанное аудио) удаляются после транскрипции.

## Integrations
- OpenAI API (Whisper для транскрипции)
- Anthropic API (Claude для саммари и генерации)
- YouTube (через yt-dlp)

## Out of scope (v1)
- Хранение истории обработанных видео
- Множественные шаблоны продуктов (только "системный промпт для агента")
- Авторизация / многопользовательский режим
- Обработка плейлистов
- Чек-листы, курсы и другие форматы продуктов

## Launch criteria
Работает на VPS, доступно через браузер. Можно вставить YouTube URL или загрузить файл, получить транскрипт, саммари и готовый промпт для агента.
